# UIDAI offline e-KYC public certificates

Verification uses the **X509 certificate embedded inside the Aadhaar XML** first (recommended by UIDAI).

These files are optional fallbacks for older XML formats:

- `uidai_offline_publickey.cer` — https://uidai.gov.in/images/uidai_offline_publickey_26022019.cer
- `okyc-publickey.cer` — https://www.uidai.gov.in/images/okyc-publickey.cer

For the latest certificate list, see UIDAI Developer Section → Certificate Details.
