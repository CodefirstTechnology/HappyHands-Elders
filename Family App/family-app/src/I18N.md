# Internationalization (i18n)

- Use **react-i18next**: `const { t } = useTranslation()` and `t('namespace.key')`.
- Locale files: `src/locales/en.json`, `hi.json`, `mr.json`.
- Language is stored locally via **expo-secure-store** (native) / **localStorage** (web) in `src/lib/languageStorage.ts`, and synced to the user profile via `PATCH /api/v1/auth/me/preferences`.
- Changing language in **Login**, **Register**, or **Profile** updates the UI immediately (no app restart).
- Status codes: `translateStatus('PENDING')` or `t('status.PENDING')`.
- Dates/numbers: `formatDate`, `formatTime`, `formatCurrency`, `formatRelativeTime` from `@/lib/i18n/format`.
- Push/in-app notifications: `localizeNotification()` from `@/lib/i18n/notifications`.

**Do not add hardcoded user-visible strings in new screens.**
